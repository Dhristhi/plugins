# Contributing

Thanks for your interest in contributing! This project is a React + Vite app showcasing a JSON Forms-based form builder. We welcome bug fixes, enhancements, and documentation improvements.

## Getting Started

- Fork the repo and create a feature branch from `main`.
- Use Node 18+ (or the version in `.nvmrc` if present).
- Install deps and run locally:
  ```bash
  yarn install
  yarn dev
  ```

## Development

- Code style: Prefer modern React patterns. If you add linting/formatting, use ESLint + Prettier.
- UI: Keep Material UI components and design consistent.
- Keep changes focused and incremental.

## Commit Messages

- Use clear, descriptive messages. Conventional Commits are encouraged (e.g., `feat:`, `fix:`, `docs:`).

## Pull Requests

1. Ensure the app builds locally: `yarn build`.
2. Add/update documentation when relevant.
3. Include screenshots/GIFs for UI changes when possible.
4. Link related issues and describe the motivation and approach.

## Issue Reports

- Use the Bug or Feature templates.
- Include steps to reproduce, expected vs. actual behavior, and environment details.

## Security

- Please do not open public issues for security matters. See [SECURITY.md](SECURITY.md).
