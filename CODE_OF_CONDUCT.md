# Code of Conduct

We follow the Contributor Covenant Code of Conduct, version 2.1.

- Standard text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
- FAQ: https://www.contributor-covenant.org/faq

By participating in this project, you agree to:
- Be respectful and inclusive.
- Use welcoming, professional language.
- Gracefully accept constructive criticism.
- Focus on what is best for the community.

If you witness or experience unacceptable behavior, report it to the maintainers as described in SECURITY.md.

This Code of Conduct is adapted from the Contributor Covenant, version 2.1.
